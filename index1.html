// features

Built for Real Use

Not a gimmick — full OS-level control built for daily use.

Full Mouse Control
Move, click, right click, double click, scroll, drag — all with hand gestures under 30ms latency.
Context Aware
Swipe means "back" in Chrome, "next slide" in PowerPoint, "next track" in Spotify. Adapts automatically.
Voice + Gemini AI
30+ offline commands built in. Anything else handled by Gemini AI — open apps, type text, run shortcuts.
Ultra Low Latency
Win32 SendInput bypasses all OS overhead. Dual-EMA smoothing kills jitter while keeping cursor crisp.
Any Webcam Works
Built-in laptop cams, USB webcams, even low-res 480p cameras. No special hardware required.
Minimal UI
Tiny corner HUD, slide-in sidebar, system tray. AIROS runs completely invisible while you work.
