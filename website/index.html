Everything You Need

Full mouse and keyboard replacement, built on modern AI.

🖱️
Full Mouse Control
Move, click, right click, double click, scroll, drag — all with hand gestures in real time at under 30ms latency.
🧠
Context Aware
Swipe means "back" in Chrome, "next slide" in PowerPoint, "next track" in Spotify. Adapts automatically to active app.
🎙️
Voice + Gemini AI
30+ built-in commands offline. Anything else sent to Gemini AI — open apps, type text, run shortcuts naturally.
⚡
Ultra Low Latency
Win32 SendInput bypasses all overhead. Dual-EMA smoothing kills jitter while keeping cursor response crisp.
💤
Smart Idle
Auto-sleeps when no hand detected. Wake instantly with open palm. Never blocks your real keyboard or mouse.
🪟
Minimal UI
Tiny corner HUD, slide-in sidebar, system tray. Completely invisible while you work.
